# The Roulette Deck

Brutalist arcade chaos sorter for iPad-sized screens.

## What it does

- Stores your brain-dump entries in local browser storage
- Spins between your chaos inbox and built-in daily vibes
- Lets you archive, save, or lock in a card for one hour
- Includes wildcard mode, retro audio chirps, and swipe gestures

## Privacy

Your personal entries are stored in your browser with `localStorage`.
They are not bundled into this repository and are not uploaded by the app.

## Hosting

This repo is set up to work as a static site on GitHub Pages.
